# Backend (MySQL Version)

**Note:** This backend is for the MySQL version of the project. The current deployed version uses LocalStorage and doesn't require this backend.

## Setup (Optional)

If you want to use the MySQL version:

1. Install dependencies:
```bash
cd backend
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Update `.env` with your MySQL credentials

4. Start server:
```bash
npm start
```

## Current Version

The project currently uses:
- Frontend-only architecture
- LocalStorage for data
- No backend required
- Deployed on Render with static server

This backend is kept for reference only.
