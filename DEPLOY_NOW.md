# ✅ Render Web Service - Quick Steps

## 🚀 Deploy Now (Follow These Steps)

### 1. Go to Render
👉 https://render.com

### 2. Sign In
- Click "Sign In"
- Choose "Sign in with GitHub"

### 3. Create Web Service
- Click **"New +"** (top right)
- Select **"Web Service"**

### 4. Connect Repository
- Find: `student_success_prediction_system`
- Click **"Connect"**

### 5. Fill Settings

**Copy these exactly:**

```
Name: student-success-system
Environment: Node
Branch: main
Root Directory: (leave empty)
Build Command: npm install
Start Command: npm start
```

**Instance Type**: Select **"Free"**

### 6. Create
- Click **"Create Web Service"**
- Wait 2-3 minutes ⏳

### 7. Done! 🎉
Your URL: `https://student-success-system.onrender.com`

---

## ⚠️ Important

**First Load**: Takes 30-60 seconds (app waking up)
**After 15 min idle**: App sleeps, next load takes 30-60 sec
**Solution**: Use UptimeRobot (free) to keep it awake
